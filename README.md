# GeoQuest

GeoQuest is a location-based Android application that allows users to create and complete real-world quests using geolocation tracking and map integration.

The project was developed as part of a collaborative university software engineering project and focuses on interactive mobile experiences, user-generated content, and location-aware gameplay systems.

---

## Features

- Create custom location-based quests
- View quests using interactive map functionality
- Track user location in real time
- Complete quests by visiting physical locations
- User account/authentication support
- Persistent quest and user data storage
- Mobile-first Android UI/UX

---

## Technologies Used

- Android Studio
- Java
- Firebase
- Google Maps API
- Geolocation Services
- RecyclerViews / Fragments
- Git & GitHub


## Installation

1. Clone the repository

```bash
git clone https://github.com/forrestgraycode/GeoQuest.git
```

2. Open the project in Android Studio

3. Configure Firebase credentials if necessary

4. Build and run the application on an emulator or Android device
